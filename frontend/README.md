# frontend

Next.js 15 + React 19 + Tailwind + TypeScript app for BakeryPilot. See the repo
README for context.

## Structure

- `src/app/` -- pages: chat, materials, schedule, scorecard, facilities
- `src/components/` -- ActionCard, SupplierCard, FlowSightCanvas, YieldCounter, ...
- `src/lib/api.ts` -- typed HTTP client for all backend endpoints

## Run

```bash
npm install
npm run dev
```
